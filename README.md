# console-garland

> [!IMPORTANT]
> *repo is in lazy progress. this is not a final product.*


-●-●-●- Цветная консольная гирлянда, реализованная на Python с использованием библиотеки `colorama`. -●-●-●-

Простенький проект был написан для создания небольшой новогодней атмосферы в вашем оконном (тайловом) менеджере.

Имеется множество режимов, _горячие клавиши_ для управления гирляндой и описание режима.
